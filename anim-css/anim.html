Transformation-Animation

0% { transform: scale(1); }

25% { transform: scale(1.3); }

50% { transform: rotate(45deg); background: tomato; }

100% { transform: rotate(0); } }

.box-anim { animation: pop 4s linear 1s 3 alternate; }
